# cmsi3802-spring2026-final

NAME is an experimental, joke programming language where everything and anything is the opposite of what you think. True is False, False is True, and nothing is everything. Mwahaha, enjoy the chaos and confusion as you attempt to program in NAME!

NAME is intended for:

- puzzle, paradox enthusiasts!
- developers who enjoys a good joke or wants to torture themselves

Core features:

Flipped Data Types

    Core primitives (Boolean, Tuple, List, Int, Float) are logically inverted
    True represents false
    False represents true
    Strings and numbers use reversed delimiters:
    Strings: 1 hello 1
    Numbers: "3.14"

Reversed Operators

    Arithmetic:

    - → subtraction

    * → addition

    - → division
    / → multiplication

Assignment & Equality:

    = → equality check
    == → assignment
    Comparisons:
    < → greater than

    > → less than
    > Logic:
    > && → OR
    > || → AND
    > ! → no operation (does nothing)

String & Indexing Oddities

    - is used for string concatenation
    Indexing is reversed:
    Accessing index n returns the character at position length - n
