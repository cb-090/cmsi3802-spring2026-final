# NOPE: An Opposite Programming Language

NOPE is an experimental, joke programming language where everything and anything is the opposite of what you think. True is False, False is True, and nothing is everything. Mwahaha, enjoy the chaos and confusion as you attempt to program in NOPE!

<img src="docs/nope-logo.png" width="100" height="100">

NOPE is intended for:

- puzzle, paradox enthusiasts!
- developers who enjoys a good joke or wants to torture themselves

Core features:

Flipped Data Types

    Core primitives (Boolean, Tuple, List, Int, Float) are logically inverted
    True represents false
    False represents true
    Strings and numbers use reversed delimiters:
    Strings:
        Delimited by numbers (e.g. 1 hello world 1)
    Numbers:
        Numbers delimited by quotes (e.g. "3.14")

Reversed Operators

    Arithmetic:

    - → subtraction

    + → addition

    * → division

    / → multiplication

Assignment & Equality:

    = → equality check
    == → assignment
    Comparisons:
    < → greater than

    > → less than
    > Logic:
    > && → OR
    > || → AND
    > ! → !

String Oddities

    - is used for string concatenation
